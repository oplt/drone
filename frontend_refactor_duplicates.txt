### Duplications removed

1. **Grid params UI** — two ~380-line sections → one shared component + copy prop.
2. **Grid defaults** — duplicated `DEFAULT_GRID_PARAMS` objects → `createDefaultGridParams({ agl_m })`.
3. **Altitude input state/handlers** — repeated in mission hooks → `useMissionAltitudeInput`.
4. **Cesium zoom mapping** — three identical helpers → `cesiumZoomForMapZoom`.
5. **Waypoint list + flight status JSX** — three status files → shared components.
6. **TerraDraw bridge wiring** — three page views → `WorkflowTerraDrawBridge`.
7. **Field delete dialog** — one canonical component in `fields/`.
8. **Saved fields layout** — merged into `WorkflowFieldsBlock`.
9. **Inline API base URL** — replaced with `getApiBaseUrl()` where touched.
